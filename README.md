# BMI Calculator with Body Type Analysis

A comprehensive Body Mass Index (BMI) calculator web application built with React and TypeScript. This application provides real-time BMI calculations, body type analysis, and personalized health insights.

## Features

- **BMI Calculator**
  - Real-time BMI calculation
  - Support for both metric and imperial units
  - Visual BMI category feedback
  - Personalized health recommendations

- **Body Type Quiz**
  - Interactive questionnaire
  - Visual feedback
  - Personalized body type analysis
  - Custom recommendations based on body type

- **Progress Tracking**
  - BMI history visualization
  - Progress charts
  - Exportable reports

- **Modern UI/UX**
  - Responsive design
  - Dark/Light theme support
  - Clean and intuitive interface

## Tech Stack

- React with TypeScript
- Tailwind CSS
- Shadcn/ui for UI components
- Recharts for data visualization
- HTML2Canvas for report generation

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
